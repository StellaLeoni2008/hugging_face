## 📌 About

This repository contains a collection of AI applications developed during the **Alura – Hugging Face: Exploring and Applying AI Solutions with Foundation Models** course.

Throughout the course, several practical projects were built using **pre-trained Hugging Face models** to solve Natural Language Processing (NLP) and Computer Vision tasks. Rather than training models from scratch, the applications leverage the **Transformers** ecosystem to integrate state-of-the-art AI models into interactive Python applications.

Each folder represents an independent project, demonstrating a different real-world use case ranging from sentiment analysis and product categorization to image processing, intelligent question answering, and conversational AI.

---

## 📚 Repository Structure

### 📁 01. Sentiment Analysis & Product Categorization
This application explores two common NLP tasks using Hugging Face models.

The first module performs **Sentiment Analysis** on customer reviews, while the second automatically categorizes products using **Zero-Shot Classification**, allowing products to be classified without training a dedicated model.

**Main topics explored**
- Sentiment Analysis
- Zero-Shot Classification
- Natural Language Processing
- Customer Review Analysis
- Product Categorization
- Hugging Face Pipelines

---

### 📁 02. Image Background Removal
Computer Vision application capable of automatically removing the background from uploaded images using a pre-trained segmentation model from Hugging Face.

**Main topics explored**
- Computer Vision
- Image Segmentation
- Background Removal
- Image Processing
- Hugging Face Models
- Gradio Interface

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/StellaLeoni2008/hugging-face-ai-applications.git
cd hugging-face-ai-applications
```

### 2. Create a virtual environment

```bash
python -m venv venv
```

### 3. Activate the virtual environment

**Windows**
```bash
venv\Scripts\activate
```

**macOS/Linux**
```bash
source venv/bin/activate
```

### 4. Install the required libraries

The notebooks contain the necessary installation commands for their respective dependencies.

Open the desired notebook and execute the installation cells before running the rest of the project.
